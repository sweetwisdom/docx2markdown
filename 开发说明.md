# 安装

uv venv --python 3.11

uv pip install -r requirements.txt



## 本地测试1

```sh

# 1. 初始化 uv 项目（会创建 pyproject.toml）
uv init --no-readme

# 2. 添加依赖
uv add python-docx lxml

# 3. 安装项目本身（开发模式）
uv pip install -e .

# 4. 运行
uv run docx2markdown demo/test-text.docx test-output.md

uv run docx2markdown dist/aaa.docx dist/aaa.md

# 卸载
 
uv pip uninstall docx2markdown

```
